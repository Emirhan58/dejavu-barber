       - Static import'u `next/dynamic` ile degistir:
         ```typescript
         import dynamic from "next/dynamic";
         const IntroOverlay = dynamic(() => import("@/components/animations/IntroOverlay").then(m => m.IntroOverlay), { ssr: false });
         ```
