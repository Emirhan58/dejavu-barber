## Spacing Scale

Declared values (all multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, tight stacking |
| sm | 8px | Compact element spacing, input padding |
| md | 16px | Default element spacing, card padding, navbar item gaps |
| lg | 24px | Section inner padding (mobile), card content padding |
| xl | 32px | Layout gaps between components |
| 2xl | 48px | Section vertical padding (mobile: --spacing-section-mobile maps to 3rem) |
| 3xl | 64px | Page-level spacing, hero content offset |
| 4xl | 96px | Section vertical padding (desktop: --spacing-section maps to 6rem) |

Exceptions:
- Touch targets on mobile: minimum 44x44px hit area for all tappable elements (navbar links, menu items, WhatsApp CTA)
- Navbar height: 64px (desktop), 56px (mobile)
- Full-width sections: 0px horizontal padding on section wrappers; content constrained by max-width 1200px (--container-content) with 16px horizontal padding

**Source:** Claude's discretion (CONTEXT.md delegated exact spacing scale)

### Accent Reserved For (neon-red #E11D48)

These elements and ONLY these elements receive neon-red accent color or neon glow effect:

1. **DEJAVU logo** in navbar — neon glow text-shadow
2. **CTA buttons** ("Randevu Al") — background-color or border + glow
3. **Active scroll-spy indicator** — neon underline below active nav item
4. **Hero title** — neon glow text-shadow on DEJAVU text
5. **Barber pole divider stripes** — red stripe in red-white-blue pattern
6. **WhatsApp floating button** hover state — neon-red ring

Neon glow is NOT applied to: service cards, footer/contact section, normal links, body text, icons, gold accent elements.

## Visual Motifs

### Grain Overlay
- **Technique:** CSS pseudo-element (::before) on body or root wrapper
- **Opacity:** 0.04 (3-5% range, using 4%)
- **Position:** fixed, inset: 0, z-index: 50, pointer-events: none
- **Pattern:** Inline SVG feTurbulence (fractalNoise, baseFrequency 0.65, numOctaves 3)

### Barber Pole Section Divider
- **Height:** 8px (h-2)
- **Pattern:** repeating-linear-gradient at 120deg — #E11D48 (red), #FAFAF9 (white), #3B82F6 (blue)
- **Stripe width:** 20px each color
- **Animation:** Horizontal slide (barber-slide keyframe), subtle, optional per tier

### Retro Card Frame
- **Border:** 1px solid rgba(212, 168, 83, 0.3) — gold at 30% opacity
- **Corner decorations:** ::before (top-left) and ::after (bottom-right) pseudo-elements
- **Corner size:** 20px x 20px
- **Corner border:** 2px solid #D4A853 (gold)

### Neon Text CSS Classes
- **.neon-glow:** text-shadow with 4 layers — 7px, 10px, 21px, 42px spread at decreasing opacity
- **.neon-glow-subtle:** text-shadow with 2 layers — 5px and 15px at low opacity

**Source:** CONTEXT.md locked decisions (grain, dividers, card frames, neon reserved usage), RESEARCH.md code examples

## Animation Tier Contract

| Tier | Condition | Allowed |
|------|-----------|---------|
| Tier 1 (Full) | Desktop >= 768px AND no prefers-reduced-motion | Parallax, particles, complex scroll sequences, neon glow animations, all Motion effects |
| Tier 2 (Reduced) | Mobile < 768px AND no prefers-reduced-motion | Simple fade-in/slide-up, navbar transition, menu animation, hover-to-tap adaptation |
| Tier 3 (None) | prefers-reduced-motion: reduce (any screen) | No animations, instant transitions, static layout |

**Detection:** useAnimationTier() hook — CSS matchMedia for width (768px breakpoint) + prefers-reduced-motion. Reduced-motion is highest priority override.

**Smooth scroll:** Lenis active only in Tier 1. Mobile and Tier 3 use native scroll.

**Source:** CONTEXT.md locked decisions (all tier definitions specified)
