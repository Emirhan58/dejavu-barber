## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, icon-to-label gap |
| md | 16px | Default element spacing, card inner padding, masonry column-gap |
| lg | 24px | Card padding, form field spacing |
| xl | 32px | Section content inner padding, grid gap |
| 2xl | 48px (--spacing-section-mobile: 3rem) | Section vertical padding on mobile |
| 3xl | 96px (--spacing-section: 6rem) | Section vertical padding on desktop |

Exceptions:
- WhatsApp floating button: 24px from viewport edges (bottom-6 right-6)
- Hero section: full viewport height (min-h-screen), no fixed spacing
- Touch target minimum: 44px for WhatsApp button (w-14 h-14 = 56px, exceeds minimum)

Source: globals.css --spacing-section / --spacing-section-mobile tokens.

## Typography

| Role | Size | Weight | Line Height | Font | Usage |
|------|------|--------|-------------|------|-------|
| Body | 16px (text-base) | 400 (normal) | 1.5 | Inter | Paragraphs, descriptions, service card text, contact info |
| Label | 14px (text-sm) | 700 (bold) | 1.4 | Inter | Filter buttons, category labels, working hours, small annotations |
| Heading | 24px (text-2xl) | 700 (bold) | 1.2 | Playfair Display | Section titles (Hakkimizda, Hizmetler, Iletisim), category headings |
| Display | 36px mobile / 48px desktop (text-4xl / text-5xl) | 700 (bold) | 1.1 | Playfair Display | Hero slogan only |

Additional rules:
- CTA buttons: Inter 700, 14px, uppercase, letter-spacing 0.15em (tracking-widest)
- Service card title: Inter 700, 16px (text-base font-bold)
- Copyright line: Inter 400, 12px (text-xs)
- Neon glow class (.neon-glow) applied ONLY to: DEJAVU logo, hero slogan, scroll-down arrow. NOT on section headings, service cards, or footer.

Source: Phase 1 CONTEXT.md typography decisions, layout.tsx font weights.

## Section Layout Contracts

### Hero Section (CONT-01)
- Height: min-h-screen, flex centering
- Background: next/image fill + priority + object-cover (NOT CSS background-image) with `/images/hero/dejavu-kuafor.jpg`
- Overlay: absolute inset-0, bg-black/75 (75% opacity — within user's 70-80% range)
- Content stack (centered, z-10):
  1. Logo component: mobile h-[60px], desktop md:h-[120px]
  2. Slogan: "Sac ve Sakal Tasarimi" — display font, text-lg md:text-xl, cream color, tracking-wide
  3. CTA button: "RANDEVU AL" — links to WhatsApp with pre-filled message
  4. ScrollDownArrow component at bottom
- CTA button style (Claude's discretion decision): **Neon red outline** — border-2 border-neon-red, text-neon-red, uppercase, tracking-widest, hover:bg-neon-red hover:text-white transition-colors. Rationale: neon red aligns with the neon sign aesthetic and creates maximum contrast on the dark overlay.

### About Section (CONT-02)
- ID: #hakkimizda
- Background: base-medium (#211812) — alternating from hero
- Layout: 2-column grid (lg:grid-cols-2), gap-xl (32px)
- Left column: Section heading "Hakkimizda" (Playfair 700, text-2xl) + paragraph text (Inter 400, text-base, line-height 1.5, cream)
- Right column: next/image of kuafor-icerisi.jpg, rounded-lg, object-cover, aspect-[4/3]
- Mobile: Single column (metin ustte, foto altta)
- Paragraph text (Claude's discretion): Warm, professional introduction about Salon Dejavu in Susehri, traditional barbering meets modern style. 3-4 sentences.

### Services Section (CONT-03)
- ID: #hizmetler
- Background: base-dark (#1C1410) — alternating
- Section heading: "Hizmetlerimiz" (Playfair 700, text-2xl, centered)
- Organization: Category groups, each with:
  - Category subheading: Inter 700, text-lg, gold color, uppercase, tracking-wide
  - Grid of ServiceCards: lg:grid-cols-3 md:grid-cols-2 grid-cols-1, gap-md (16px)
- Categories: Sac, Sakal, Cilt Bakimi, Ozel Hizmetler
- ServiceCard anatomy:
  - RetroCard wrapper (existing gold corner ornaments)
  - Vintage SVG icon (40x40px, cream color, centered top)
  - Service name: Inter 700, text-base
  - Service description: Inter 400, text-sm, text-secondary (70% cream opacity), 1-2 sentences
  - NO price displayed

### Gallery Page (GALR-01, GALR-02)
- Route: /galeri
- Background: base-dark (#1C1410)
- Page heading: "Galeri" (Playfair 700, text-3xl, centered)
- Filter bar: Horizontal row of filter buttons, centered, gap-sm (8px)
  - Filter button inactive: Inter 700, text-sm, cream/50 opacity, px-4 py-2
  - Filter button active: cream full opacity + neon-red bottom border (2px)
  - Filters: "Tumu" (default) | "Dukkan" | "Calismalarimiz"
- Masonry grid: CSS columns (1 col mobile, 2 col sm:640px, 3 col lg:1024px), column-gap 16px
- Each grid item: break-inside-avoid, margin-bottom 16px, rounded-lg overflow-hidden, cursor-pointer
- Images: next/image with width/height from constants, object-cover, lazy loading (NOT priority)
- Click: Opens Lightbox at clicked image index

### Lightbox (GALR-01 detail)
- Overlay: fixed inset-0 z-50, bg-black/90
- Image: max-h-[90vh] max-w-[90vw], object-contain, centered
- Navigation: Left/right arrows (absolute positioned, cream/white, text-4xl), clickable areas
- Close: X button (absolute top-4 right-4, cream/white)
- Keyboard: Escape = close, ArrowRight = next, ArrowLeft = prev
- Body scroll: Locked when open (overflow: hidden), restored on close/unmount
- Click outside image: Closes lightbox

### Contact Section (CONT-04)
- ID: #iletisim
- Background: base-light (#150F0B) — darkest variant, footer feel
- Section heading: "Iletisim" (Playfair 700, text-2xl)
- Layout (Claude's discretion decision): **Side-by-side** — lg:grid-cols-2
  - Left: Google Maps iframe (width 100%, height 400px, rounded-lg, loading="lazy")
  - Right: Business info stack:
    - Address: with map-pin icon prefix
    - Phone: with phone icon prefix, clickable tel: link
    - Working hours: weekdays + sunday, with clock icon prefix
    - WhatsApp CTA: Secondary link "WhatsApp ile yazin" with WhatsApp icon
- Mobile: Single column (map ustte, info altta)
- Copyright: Full-width at very bottom, text-xs, text-center, cream/50 opacity, py-md (16px)
- SectionDivider component above this section

### WhatsApp Floating Button (CONT-05)
- Position: fixed bottom-6 right-6 z-40
- Shape: rounded-full, w-14 h-14 (56px — exceeds 44px touch target)
- Color: bg-[#25D366] (official WhatsApp green)
- Icon: White WhatsApp SVG, w-7 h-7
- Shadow: shadow-lg
- Hover: scale-110 transition-transform
- Link: `wa.me/905397256886?text=Merhaba%2C%20randevu%20almak%20istiyorum.`
- Accessibility: aria-label="WhatsApp ile randevu al"
- Placement: layout.tsx, after main, visible on all pages
