## Animation Timing Contract

### Intro Sequence (ANIM-01)

| Phase | Animation | Duration | Ease | Delay |
|-------|-----------|----------|------|-------|
| 1. Letter reveal | opacity 0->1, text-shadow transparent->neon | 0.2s per letter | power2.in | stagger 0.15s, random order |
| 2. Full glow pulse | text-shadow intensifies on all letters | 0.4s | power2.out | after letter reveal completes |
| 3. Slogan fade-in | opacity 0->1, y 10px->0 | 0.5s | power2.out | after glow pulse |
| 4. Hold | pause | 0.3s | - | after slogan |
| 5. Curtain up | yPercent 0->-100 (overlay slides up) | 0.8s | power3.inOut | after hold |
| **Total** | | **~2.7s** | | |

- Skip: click/tap anywhere calls `timeline.progress(1)` (jumps to end instantly)
- Repeat prevention: `sessionStorage.setItem("intro-seen", "1")` on timeline complete
- Gate: `sessionStorage.getItem("intro-seen")` checked on mount; if set, overlay never renders
- Tier behavior: Shown on ALL tiers (full, reduced). Only "none" tier skips entirely.
- Z-index: 100 (above grain overlay z-50, above everything else)

### Scroll Reveal (ANIM-02)

| Property | Value |
|----------|-------|
| Transform | y: 40px -> 0 (slide up) |
| Opacity | 0 -> 1 |
| Duration | 0.8s |
| Ease | power2.out |
| Trigger | element top reaches 80% of viewport (start: "top 80%") |
| toggleActions | play none none none (play once, no reverse) |
| Stagger (when enabled) | 0.12s between child elements |
| Stagger selector | `.scroll-reveal-item` class on children |

Applied to sections:
- About (content block, stagger on paragraphs)
- Services (container, stagger on service cards via `.scroll-reveal-item`)
- Contact (content block, stagger on info items)
- Gallery grid (stagger on grid items)

NOT applied to Hero (already visible on load, handled by intro sequence).

### Parallax (ANIM-03)

| Property | Value |
|----------|-------|
| Target | `.hero-bg-image` (Hero background Image element) |
| Effect | yPercent: 0 -> -15 (image moves up 15% over scroll) |
| Ease | none (linear, scroll-linked) |
| Scrub | 0.5 (smooth 0.5s lag behind scroll position) |
| Trigger start | top top (when hero top hits viewport top) |
| Trigger end | bottom top (when hero bottom hits viewport top) |
| Tier gate | "full" only (desktop). Reduced/none: no parallax. |

### Smoke/Steam Particles (ANIM-04)

| Property | Value |
|----------|-------|
| Particle count | 10 |
| Container | absolute, bottom 0, left/right 0, height 40% of hero, overflow hidden, pointer-events none |
| Particle size | 60-100px (randomized per particle) |
| Particle shape | border-radius 50% (circle) |
| Particle fill | radial-gradient(circle, rgba(255,255,255,0.15) 0%, transparent 70%) |
| Blur | filter: blur(20px) applied as static style (not animated) |
| Animation name | smoke-rise |
| Animation duration | 5-8s (randomized per particle) |
| Animation delay | 0-6s staggered (particle_index * 0.6s) |
| Animation timing | ease-out |
| Animation iteration | infinite |
| Keyframes 0% | transform: translate3d(0, 0, 0) scale(0.8); opacity: 0 |
| Keyframes 20% | opacity: 0.15 |
| Keyframes 100% | transform: translate3d(20px, -120px, 0) scale(1.5); opacity: 0 |
| GPU hint | will-change: transform, opacity |
| Horizontal spread | particles distributed 10%-90% across container width (left: 10 + i*8 %) |
| Scroll behavior | Container fades out when hero leaves viewport (CSS or ScrollTrigger opacity) |
| Tier gate | "full" only. Reduced/none: component returns null (zero DOM nodes). |

### Neon Glow Hover (ANIM-06)

| Element | Trigger | Effect | Duration | Ease |
|---------|---------|--------|----------|------|
| DEJAVU logo (navbar) | mouseenter | text-shadow: 0 0 7px #E11D48, 0 0 10px #E11D48, 0 0 21px #E11D48 | 0.25s | ease (CSS) |
| CTA buttons | mouseenter | text-shadow: same as above | 0.25s | ease (CSS) |
| Hero heading | mouseenter | text-shadow: same as above | 0.25s | ease (CSS) |
| Active scroll-spy line | active state | text-shadow: same as above | 0.25s | ease (CSS) |

Implementation: CSS class `.neon-hover-glow` with `transition: text-shadow 0.25s ease` and `:hover` state. Already partially exists in globals.css as `.neon-glow` and `.neon-glow-subtle`.

### Service Card Hover (ANIM-06 subset)

| Property | Value |
|----------|-------|
| Transform | y: 0 -> -3px (subtle lift) |
| Box shadow | 0 8px 30px rgba(0,0,0,0.4) |
| Duration | 0.2s |
| Ease | easeOut |
| Implementation | Motion whileHover prop |
| Tier gate | "none" tier: no hover effect. "reduced" and "full": hover active. |

NO neon glow on service cards. Only translateY + shadow deepening.

### Page Transition (GALR-03)

| Property | Value |
|----------|-------|
| Type | Fade out -> Fade in (opacity crossfade) |
| Duration | 0.3s |
| Implementation | Motion AnimatePresence mode="wait" with FrozenRouter |
| Wrapper location | layout.tsx, wrapping `{children}` |
| Key | useSelectedLayoutSegment() return value |
| Initial | false (no animation on first page load) |
| Tier gate | All tiers. "none" tier: duration set to 0 (instant). |

## Interaction States

### Intro Overlay States

| State | Visual | Behavior |
|-------|--------|----------|
| Loading (pre-check) | Hidden | sessionStorage check runs |
| Playing | Full-screen black overlay, z-100, letters animating | Blocks scroll, click/tap triggers skip |
| Skipped/Complete | Slides up (yPercent -100), then removed from DOM | Scroll unlocked, hero revealed |
| Returning visit | Never rendered | sessionStorage "intro-seen" found |
| prefers-reduced-motion | Never rendered | "none" tier skips intro entirely |

### Scroll Reveal States

| State | Visual |
|-------|--------|
| Before trigger | Element at opacity 0, y +40px (invisible below final position) |
| Animating | opacity 0->1, y 40->0, 0.8s power2.out |
| After trigger | Element at opacity 1, y 0 (final position, no further changes) |
| prefers-reduced-motion | Element visible immediately, no transform |

### Smoke Particle States

| State | Visual |
|-------|--------|
| Desktop (full tier) | 10 particles cycling through smoke-rise keyframes |
| Hero in view | Particles visible with smoke-rise animation |
| Hero scrolled past | Particles fade out (container opacity -> 0) |
| Mobile (reduced tier) | null render, zero DOM nodes |
| prefers-reduced-motion | null render, zero DOM nodes |

## Performance Guardrails

| Metric | Target | Enforcement |
|--------|--------|-------------|
| Smoke particles | Max 12 DOM nodes | Hardcoded array length in SmokeParticles component |
| CSS animations | transform + opacity only in @keyframes | No color, width, height, or filter in keyframes |
| will-change | Applied only to smoke particles | Remove from other elements to avoid layer promotion overhead |
| ScrollTrigger instances | Max 5 (about, services, contact, gallery, hero parallax) | One ScrollReveal per section, not per element |
| GSAP timeline cleanup | useGSAP handles cleanup automatically | Never use raw gsap.timeline() outside useGSAP |
| Mobile DOM | Zero smoke particle nodes on reduced/none tier | Conditional render returns null, not display:none |
| Intro overlay DOM | Removed after completion via setShow(false) | Not just hidden, actually unmounted |
