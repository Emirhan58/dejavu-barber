    **2. Create `src/components/animations/IntroOverlay.tsx`:**

    - "use client" directive
    - Import `useRef, useState, useEffect, useCallback` from react
    - Import `gsap, useGSAP` from `@/lib/gsap-config`
    - Import `useAnimationTier` from `@/hooks/useAnimationTier`
    - Component receives `children: React.ReactNode`
    - State: `show` starts as `false` (SSR-safe)
    - `useEffect` on mount: check `sessionStorage.getItem("intro-seen")` -- if found, keep `show=false` and return. Also check tier: if tier is "none", keep `show=false`. Otherwise set `show=true`.
    - NOTE on tier: Intro plays on BOTH "full" and "reduced" tiers (desktop AND mobile). Only "none" (prefers-reduced-motion) skips.
    - `skip` callback: calls `tlRef.current?.progress(1)` to jump timeline to end
    - `useGSAP` with scope `overlayRef`, dependencies `[show]`:
      - If `!show` return early
      - Get letters: `gsap.utils.toArray(".intro-letter", overlayRef.current!)`
      - Shuffle array: `[...letters].sort(() => Math.random() - 0.5)`
      - Create timeline `tl` with `onComplete`: `sessionStorage.setItem("intro-seen", "1")`
      - Phase 1 - Letter reveal: `tl.from(shuffled, { opacity: 0, textShadow: "0 0 0px transparent", duration: 0.2, stagger: 0.15, ease: "power2.in" })`
      - Phase 2 - Full glow pulse: `tl.to(".intro-letter", { textShadow: "0 0 10px #E11D48, 0 0 40px rgba(225,29,72,0.5)", duration: 0.4, ease: "power2.out" })`
      - Phase 3 - Slogan fade-in: `tl.from(".intro-slogan", { opacity: 0, y: 10, duration: 0.5 })`
      - Phase 4 - Hold: `tl.to({}, { duration: 0.3 })`
      - Phase 5 - Curtain up: `tl.to(overlayRef.current!, { yPercent: -100, duration: 0.8, ease: "power3.inOut" })`
      - Phase 6 - Cleanup: `tl.call(() => setShow(false))`
      - Store: `tlRef.current = tl`
    - If `!show`, return `<>{children}</>` (just pass through children)
    - Render when show=true:
      ```tsx
      <>
        <div
          ref={overlayRef}
          onClick={skip}
          className="fixed inset-0 z-[100] bg-black flex flex-col items-center justify-center cursor-pointer"
        >
          <div className="flex">
            {"DEJAVU".split("").map((letter, i) => (
              <span
                key={i}
                className="intro-letter font-display font-bold text-cream"
                style={{ fontSize: "clamp(64px, 12vw, 96px)" }}
              >
                {letter}
              </span>
            ))}
          </div>
          <p
            className="intro-slogan text-cream/80 text-base md:text-xl tracking-[0.2em] mt-4 font-body"
          >
            Sac ve Sakal Tasarimi
          </p>
        </div>
        {children}
      </>
      ```
